## 2. Scrum Events

### Sprint Planning
- Duration: 2 hours
- Activities:
  - Define sprint goal
  - Select backlog items
  - Break tasks into technical work

### Daily Scrum
- Duration: 15 minutes
- Conducted daily
- Discuss:
  - What was done yesterday
  - What will be done today
  - Blockers

### Sprint Review
- Demonstrate completed features
- Gather feedback from Product Owner

### Sprint Retrospective
- Discuss:
  - What went well
  - What didn’t go well
  - Improvements for next sprint

## 3. Scrum Artifacts

### Product Backlog
- List of all desired features and improvements

### Sprint Backlog
- Selected backlog items for the sprint

### Increment
- Working task management features delivered at sprint end
